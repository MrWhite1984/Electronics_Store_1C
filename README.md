<h1>Автоматизация магазина электроники на платформе 1С:Предприятие</h1>
<p>Учебная конфигурация, разработанная для автоматизации ключевых бизнес-процессов небольшого магазина электроники: учёта товаров, закупок, продаж и формирования отчётности.</p>
<p>Конфигурация учитывает специфику категории товаров: каждый товар может иметь модель, производителя, цвет и другие характеристики, хранящиеся в табличной части справочника.</p>
<h2>Основные возможности</h2>
<ul>
  <li>Поступление товаров от поставщиков с автоматическим расчётом суммы.</li>
  <li>Продажа товаров с интерфейсом «склад + корзина», фильтрацией по типу/производителю/цвету и валидацией остатков.</li>
  <li>Регистр накопления «Остатки товаров» — отслеживает движение товаров в реальном времени.</li>
  <li>Регистры сведений «Цены закупки» и «Цены продажи» — гибкое управление стоимостью.</li>
  <li>Печатная форма чека после проведения продажи с итоговой суммой, НДС и перечнем товаров.</li>
  <li>Разграничение ролей: директор, администратор, продавец, кладовщик — с настройкой прав доступа через подсистемы.</li>
</ul>
